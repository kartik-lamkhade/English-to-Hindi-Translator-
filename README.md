A simple **English to Hindi translation web application** built using **TensorFlow/Keras** and **Streamlit**.  
This app uses a trained **seq2seq model** to translate English sentences into Hindi in real-time.  

---

## Features

- Translate English sentences to Hindi instantly.  
- Interactive web interface using **Streamlit**.  
- CPU and GPU compatible (GPU optional).  
- Pre-trained **seq2seq Keras model** with `<start>` and `<end>` tokens.
